# 2020_Capstone-Design_Team_project
2020년 대학교 4학년 "캡스톤 디자인" 졸업 작품입니다.<br>
### 주제는 `"음성인식과 안면인식을 활용한 NUI LMS" 웹 사이트 개발` 입니다.<br>
> 'COVID-19'로 전국의 대학교 학생들이 학교를 가지 못하고, 각 학교의 **`LMS(Learning Management System)`** 을 통해
**온라인 강의를 수강하는 것**을 아이디어 삼아, `노트북의 웹캠과 내장 마이크`를 이용해 **온라인 강의 수강 중 졸음상태(sleep, sleepy, tired)를 확인**하고
**자리비움(Out of monitor)을 감시**하는 시스템을 갖춘 웹 페이지를 개발하는 것을 캡스톤 디자인 작품으로 선정하여 완성하였습니다.

- 본 `Repo` 에는 코드와 개발하면서 참고한 자료들이 포함되어있습니다.
